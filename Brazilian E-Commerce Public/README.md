# Projeto de Análise de Dados: E-commerce Brasileiro (Olist)
Este projeto faz parte da minha transição de carreira para Analista de Dados.

## 🎯 Objetivo
Analisar o comportamento de vendas, faturamento e performance de produtos utilizando SQL (SQLite) e Power BI.

## 🛠️ Tecnologias
- SQL (SQLite)
- Power BI
- Python (Pandas para limpeza inicial)

## 📈 Insights Extraídos
1. Faturamento Mensal (Identificação de sazonalidade).
2. Agrupando por Cidade e calcule a Soma do Valor dos Itens.
3. Quitidade de pedidos foram feitos em cada cidade.
4. Média do Valor do Frete por Cidade do Cliente

## 🚀 Como executar
As consultas estão disponíveis na pasta `/queries`. 
O dashboard pode ser visualizado no arquivo `.pbix`.
